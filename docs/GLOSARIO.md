# Glosario Trongkai

> Toda sigla con tooltip obligatorio en la UI (regla SUPER_PROMPT §6). Esta tabla es la fuente.

| Sigla / término | Definición |
|---|---|
| **MMPP** | Materias primas — los 5 subproductos agroindustriales que procesa Trongkai: alperujo, tomasa, pomasa, orujo de uva, levadura. |
| **PEF** | Pulsed Electric Field. Tecnología de pulsos eléctricos que rompe paredes celulares de la MMPP para liberar más aceite/licopeno/pectina sin dañarlos. Mantención cada 300 h (cambio de "cuarres electrónicos", ~1 MM CLP por par). |
| **PTEC** | Productos Tecnológicos (familia de SKUs de alto valor): Proteína Unicelular, Antioxidante, Aglomerante, Umami. Año de entrada 3-5. |
| **MS** | Materia Seca — peso de la masa una vez extraída el agua. |
| **Alperujo** | Subproducto de las almazaras de aceite de oliva (mezcla de orujo + alpechín). Humedad ~65%, aceite extraíble ~2% con PEF. |
| **Tomasa** | Subproducto de la industria del tomate (piel, pepas, pulpa). Humedad ~82% — la "peor" en términos de materia seca. Temporada Ene-Mar. |
| **Pomasa** | Subproducto de jugos de manzana/pera. Humedad ~80%. Temporada Mar-May. |
| **Orujo de uva** | Subproducto de vinificación. Humedad variable. Temporada Abr-Jun. |
| **Levadura** | Subproducto de cervecería/vinificación. Año corrido. |
| **Tricánter** | Decantador centrífugo de 3 fases (sólido, agua, aceite). Alternativa al prensado mecánico, mayor recuperación de aceite, mayor costo operacional. |
| **Cuello de botella** | (bottleneck) Etapa del proceso con menor capacidad por hora, define el throughput máximo. En Trongkai, **el secador** es el bottleneck esperado. |
| **Just-in-time** | Modelo operacional: no hay almacenamiento de MMPP, el camión llega y se procesa antes de fermentar. |
| **Maquilas** | Procesamiento por encargo de terceros — usar la planta para procesar MMPP que no es propia, cobrando por servicio. |
| **TIR** | Tasa Interna de Retorno. |
| **VAN** | Valor Actual Neto. |
| **WACC** | Weighted Average Cost of Capital — tasa de descuento. |
| **EERR** | Estado de Resultados (Profit & Loss). |
| **CapEx** | Capital Expenditure — inversiones en equipos. |
| **OpEx** | Operational Expenditure — gastos operativos. |
| **AACE** | Association for the Advancement of Cost Engineering — clases de estimación de CapEx (1=cotización firme, 5=concept screening ±50%). |
| **PD** | "Por Definir" — estado de un supuesto en planilla del cliente. |
| **OK\*** | Estado del cliente: "OK provisorio", validado pero no firme. |
| **CMF** | Comisión para el Mercado Financiero (Chile). Regulador de FIP CEHTA. |
| **FIP CEHTA ESG** | Fondo de Inversión Privado dueño de Trongkai (gestionado por AFIS). |
| **AFIS** | Administradora del FIP. |
| **B Corp** | Certificación de empresa B (ya la tienen). |
| **REP** | Ley de Responsabilidad Extendida del Productor (Chile) — relevante para valorización de residuos. |
| **Cuarres electrónicos** | Repuestos del PEF que se cambian cada 300 horas (jerga de Jaime, ~1 MM CLP el par). |
| **Sankey** | Diagrama de flujo proporcional, obligatorio para visualizar balance de masa. |
| **Modo A (Balance)** | Cálculo de rendimiento sobre la **base inicial** (masa que entra a planta). |
| **Modo B (Balance)** | Cálculo de rendimiento sobre la **base deshidratada** (materia seca post-secado). Diferencia material vs modo A (35% vs 38.5% en ejemplo alperujo). |
| **Trongkai Feed** | Marca comercial de ingredientes para nutrición animal (acuicultura + pet food). Compite contra harina de pescado y aditivos sintéticos. |
| **Trongkai Food** | Marca comercial de harinas funcionales 100% vegetales para humanos (panadería, snacks). Insumos clave: tomate y oliva. |
| **Opticept** | Marca registrada del equipo PEF (Pulsed Electric Field) usado por Trongkai. Alianza estratégica. Mantención cada 300 h. |
| **Axolot** | Tecnología de extracción/separación aliada con Trongkai (junto con Opticept). |
| **ACV** | Análisis de Ciclo de Vida — cuantifica impacto ambiental end-to-end del proceso. Credencial obligatoria para mercados ESG. |
| **SOPs** | Standard Operating Procedures — protocolos de calidad estandarizados por etapa, requeridos para acceso a mercados globales. |
| **Valorización en cascada** | Filosofía operativa: cada flujo de subproducto encuentra su valor óptimo antes de descartarse. "Residuo cero". |
| **Fraccionamiento inteligente** | Etapa del proceso que separa líquidos y sólidos del subproducto, dirigiendo cada componente a su ruta de mayor valor. |
| **Harina de pescado** | Benchmark de precio para Trongkai Feed. Insumo dominante en acuicultura mundial cuya sustitución es la tesis comercial principal. |
| **Hoja de Ruta Circular Chile 2040** | Marco regulatorio nacional que prohíbe prácticas obsoletas (quemas agrícolas) y empuja la valorización de residuos agroindustriales. |

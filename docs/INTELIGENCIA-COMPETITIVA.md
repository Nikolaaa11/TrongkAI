## 4. Regulación: Ley REP Chile + Hoja de Ruta Circular 2040

- **Ley 20.920 (Ley REP)** vigente desde 2017; régimen agroindustriales en consolidación 2025-2026.
- 2025: 6 sistemas de gestión aprobados (neumáticos primero).
- 2026: maduración del régimen, fiscalización activa.
- **Convenio Campo Limpio**: envases agroindustriales con triple lavado.
- **Definición pendiente "subproducto vs residuo"**: clave para Trongkai — si MMPP entrante se clasifica como "subproducto" en lugar de "residuo", se evitan trabas regulatorias en transporte y trazabilidad.

**Riesgos regulatorios para Trongkai**:
- Trazabilidad obligatoria de residuos: agrega costo administrativo (~5-10 MM CLP/año).
- Certificaciones requeridas para mercado export: BRC, FSSC 22000, ISO 22000 (incluido en `costos.comercializacion`).

**Oportunidades regulatorias**:
- Subsidios CORFO + ANID para innovación circular (los patines 300-400 MM CLP que mencionó Jaime).
- Ventaja competitiva con B Corp (ya tienen) + ACV en desarrollo.
- Posicionamiento como "compliance partner" para productores agro que necesitan disposición REP-compatible.
